** README-First

To use this Audio Drama Project Skeleton /(template)/ for your own projects, perform the following steps:

1. *Clone a Copy of this File:* Put it where you normally organize your audio drama or podcast projects.  *Place it within its own folder* named after your intended project title...

2. *NOTE:* You may be doing this because you have some fantastic story idea brewing in your mind and you need to write that down before you forget!  Right?  Therefore... as soon as you clone this file *STOP* and go immediately to the ~üí° Ideas Scratchpad Refile / ‚òû Start Here!~ section and read the ~:README:~ directly under that heading...  Then write your idea down quickly /(as per the README instructions)/ to get your inspiration recorded before you forget it...  

   Once that is done, come back here and finish setting up your project... 

   This project template is *HUGE!* Your inspiration is *tiny* and *vulnerable* to getting lost forever!  The *‚òû Start Here* section is your remedy... Do that first!    

3. *Change the* ~#+TITLE:~ *directive at the top of this file* to /"Your Audio Drama Project's Title"/

4. *Change the* ~#+EXPORT_FILE_NAME:~ *directive at the top of this file* to /"Your-Audio-Drama-Project's-Title.pdf"/ That will be the file name(s) produced during export thence forward /(including auto generated files: .tex, etc...)/

5. *Change the* ~#+DESCRIPTION:~ *directive at the top of this file* to /Your-Audio-Drama-Project's Description/ etc...

6. *Copy/Clone* ~./media~ /(and all its contents - currently empty but will contain media in future releases):/ *to:* your new project's folder /(where you copied this file).../  Any external documents or images referenced by your audio project should be placed within this ~./media/~ directory... External links to images, etc. within boilerplate examples of future versions of this ~.org~ file can be used as examples for getting your own images, and external attachments linked properly...  This single ~./media~ directory will serve media and external attachments for this and any other extra audio drama ~.org~ files you may end up creating here, as well as being the media repository used by LaTeX during publishing... You will also be using this ~./media/~ directory for all finished sound media, /(e.g., sound-effects, loops, music, etc.)/ that are called out within the cue sheets /(also contained within this ~.org~ project)/ 

7. *Copy/Clone ./bin* /(and all its contents):/ *To:* your new project's folder /(where you copied this file).../  This is needed for running scripts that convert your ~.fountain~ scripts into PDF or other formats...

8. *Copy/Clone ~./auto/~ /(and all its contents - which may or may not be empty):/ *To:* your new project's folder /(where you copied this file)/ This is where your output ~.PDF~ files will be placed during export. 

9. *Copy/Clone* ~~./org-templates~~ /(and all its contents):/ *To:* your new project's folder /(where you copied this file).../ The files in there are automatically linked and included within this template. You don't need to change those settings unless you wish to customize the already customized tags, and export options. /They are allready set up correctly for the audio drama genre, and the category is all set as well... But you can and may modify the original files to fit your needs.../

10. *Start Planning / Writing Your Audio Drama:* /Change headings and structure to match your project's outline episodes-scenes-plots-characters, etc./ Start doing the Tasks within the Tasks section... Have fun doing this...

  The output can be customized using configuration file passed as --config parameter. Config should be a JSON with this structure:

  #+begin_src js
  {
     "embolden_scene_headers": false,
     "show_page_numbers": true,
     "split_dialogue": true,
     "print_title_page": true,
     "print_profile": "a4",
     "double_space_between_scenes": false,
     "print_sections": false,
     "print_synopsis": false,
     "print_actions": true,
     "print_headers": true,
     "print_dialogues": true,
     "number_sections": false,
     "use_dual_dialogue": true,
     "print_notes": false,
     "print_header": "",
     "print_footer": "",
     "print_watermark": "",
     "scenes_numbers": "none",
     "each_scene_on_new_page": false
  }
  #+end_src

**** Available Configuration Options:

|-----------------------------+-------------------------+----------------------------------|
| *Option*                      | *Value*                   | *Description*                      |
|-----------------------------+-------------------------+----------------------------------|
| embolden_scene_headers      | true/false              | -                                |
| show_page_numbers           | true/false              | -                                |
| split_dialogue              | true/false              | Split dialogue between pages?    |
| print_title_page            | true/false              | -                                |
| print_profile               | a4 or usletter          | paper size                       |
| double_space_between_scenes | true/false              | -                                |
| print_sections              | true/false              | print sections (marked with #)   |
| print_synopsis              | true/false              | print synopsis (market with =)   |
| print_actions               | true/false              | print action blocks              |
| print_headers               | true/false              | print scene headers              |
| print_dialogues             | true/false              | print dialogues                  |
| number_sections             | true/false              | auto-numbering sections          |
| use_dual_dialogue           | true/false              | print dual dialogue in 2 columns |
| print_notes                 | true/false              | print notes                      |
| print_header                | string                  | text to put on the top           |
| print_footer                | string                  | text to put on the bottom        |
| print_watermark             | string                  | watermark text                   |
| scenes_numbers              | none, left, right, both | side of auto-numbering scenes    |
| each_scene_on_new_page      | true/false              | break page after a scene         |
